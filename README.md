# blockchain
타입스크리트로 만든 블록체인 강의 따라하기

## 실행환경
vscode, window

## install
nods.js

## intstall in terminal
npm 패키지 설치
```
npm init -y
```
타입스크립트 설치
```
npm i -D typescript
```
설정 파일생성 <br>
```
code tsconfig.js
```
저장시 자동 build를 위한 nodemon 설치
```
npm i -D ts-node
```
```
npm i nodemon
```

## 실행방법
```
npm run dev
```
